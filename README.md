# 🤟 Распознавание жестов рук (цифры 0-9)

Веб-приложение для распознавания жестов рук в реальном времени с использованием машинного обучения и компьютерного зрения.

## 📋 Описание

Проект использует MediaPipe для детекции ключевых точек руки и модель Random Forest для классификации жестов, представляющих цифры от 0 до 9. Веб-интерфейс построен на Flask и позволяет распознавать жесты через веб-камеру в реальном времени.

## ✨ Возможности

- 📹 Распознавание жестов рук в реальном времени через веб-камеру
- 🔢 Классификация жестов для цифр от 0 до 9
- 🌐 Веб-интерфейс для удобного использования
- 🎯 Высокая точность распознавания благодаря Random Forest
- 📊 Визуализация распознанных жестов на видеопотоке

## 🛠️ Технологии

- **Python 3.x**
- **Flask** - веб-фреймворк
- **OpenCV** - обработка видео и изображений
- **MediaPipe** - детекция ключевых точек руки
- **scikit-learn** - машинное обучение (Random Forest)
- **NumPy** - работа с массивами данных

## 🚀 Использование

### 1. Сбор данных для обучения

Запустите скрипт для сбора изображений жестов:
```bash
python collect_imgs.py
```
Следуйте инструкциям на экране для создания датасета.

### 2. Создание датасета

Обработайте собранные изображения:
```bash
python create_dataset.py
```

### 3. Обучение модели

Обучите классификатор на собранных данных:
```bash
python train_classifier.py
```

### 4. Тестирование (опционально)

Протестируйте модель через OpenCV:
```bash
python inference_classifier.py
```

### 5. Запуск веб-приложения

Запустите Flask-приложение:
```bash
python app.py
```

Откройте браузер и перейдите по адресу: `http://127.0.0.1:5000`

## 🎮 Как использовать веб-приложение

1. Запустите приложение командой `python app.py`
2. Откройте браузер и перейдите на `http://127.0.0.1:5000`
3. Нажмите кнопку "Start" для запуска камеры
4. Покажите жест руки перед камерой
5. Приложение распознает и отобразит соответствующую цифру
6. Нажмите "Stop" для остановки камеры

## 📊 Точность модели

После обучения модель показывает точность распознавания, которая выводится в консоль. Типичная точность составляет более 90% при качественном датасете.
